# gorgeous-admin-server 管理端后台

使用`Koa`、`MySQL`、`TypeORM`

## 包含功能：
✅ 登录 JWT 鉴权

✅ 用户的增删改查

✅ 错误处理

✅ 权限

## 使用说明

```
// 拉代码
git clone git@github.com:zhuhengtan/gorgeous-admin-server.git

// 安装
npm i

// 配置 /env/development !!!!!
// 配置 /env/development !!!!!
// 配置 /env/development !!!!!

// 运行
npm run dev

// 如果提示没有ts-node，要全局安装
npm i ts-node -g

```
